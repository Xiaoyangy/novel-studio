# 情感叙事与人物推进写法

来源摘要：
- Helping Writers Become Authors: scene arc 需要比较场景开始/结束的主导情绪与人物预期，若没有变化，场景容易显得太直。
- Writer's Digest: 情感真实来自人物发展、冲突、节奏、对白、视角和场景共同作用；避免单一反应，重视未说出口的东西。
- Advanced Fiction Writing: 动机-反应单元要求先给 POV 角色可感知的外部刺激，再写角色反应，读者才能同步体验。
- Cellar Door Editorial: 情绪既可以直接命名，也可以间接展示；关键是平衡和变化，不要只靠同一种身体反应。
- 中文人物分析资料: 人物言行由身份、地位、经历、教养和气质决定；人物情感可由语言、动作、心理、环境和关系共同揭示。
- 方格子对话人物刻画: 角色交流通常同时隐藏和揭露信息，过分准确直白会让对白变平。

## 1. 场景情绪弧

每个关键场景至少检查三件事：

1. 开场主导情绪：角色带着什么预期、恐惧或防御进入。
2. 中段压力变化：哪个证据、误判、打断或关系动作迫使情绪换挡。
3. 结尾残留状态：角色离开场景时多了什么债、羞耻、愧疚、保护欲或行动冲动。

不合格写法：事件发生了，人物只正确应对。
合格写法：事件改变了人物对自己、对别人或对规则的判断。

## 2. 动机-反应顺序

写情绪前先写角色能感知到的外部刺激：

- 看见：字段、红框、纸边、脸色、手势、倒计时。
- 听见：键盘声、电话铃、对话断口、风机异响。
- 触到：工牌勒手、纸张割手、手汗误触、杯沿温度。
- 失去：无法点击、无法解释、无法离开、无法说出某个名字。

然后写反应，顺序可参考：生理反应 -> 下意识动作 -> 选择/拒绝 -> 台词。不要先写抽象总结。

## 3. 人物刻画必须带身份压力

人物不是性格标签，而是身份、资源、恐惧和关系在场景压力下的反应。

- 职业决定他先注意什么。
- 地位决定他说话能不能直说。
- 经历决定他会误判什么。
- 私人边界决定他宁可牺牲什么也不说。
- 关系债决定他会为谁破例。

每个关键角色的台词至少有两层：表面要拿到什么，底下想藏住什么。

## 4. 情绪不用总是解释

可以直接命名情绪，但不能连续只命名情绪。优先用以下载体：

- 物件变化：便签多一串数字，工牌被捂热，纸角被捏皱。
- 行为偏差：点错按钮、删掉重来、话到嘴边换词。
- 关系动作：推开、挡住、递回、替对方留白。
- 环境打断：电话铃、旁人翻纸、风机噪音、门外声浪。

身体反应也要轮换，不要全章只写心跳、发抖、喉咙发紧。

## 5. 对话承载情感

对白不是把人物心里话说出来，而是让人物为了自保、试探、遮掩、求助或争夺定义权而说话。

- 允许答非所问、被打断、改口、只说半句。
- 让某句台词同时推进信息和暴露防御。
- 三人以上场景不必人人轮流发言；有人只用动作、沉默或无效碎语存在。
- 说话标签服务辨认，不服务工整；能靠声口和位置辨认时可省。

## 6. 长循环处理规则

如果同一章连续两轮以上围绕相似问题循环，不要继续凭感觉改：

1. 先定位循环类型：人物动机、情绪落点、对白摩擦、信息密度、AI 检测误判或外部平台口径。
2. 用 `craft_recall(methodology)` 检索对应写法；主题词至少包含人物刻画、情感叙事、情绪弧线、动机反应、对白潜台词中的两个。
3. 若 craft 命中弱或无料，必须 `web_research` 查写作资料或检测平台资料，转成项目规则后再改。
4. 每轮只改能落到正文的 1-3 个局部点，避免把已通过资产磨平。
