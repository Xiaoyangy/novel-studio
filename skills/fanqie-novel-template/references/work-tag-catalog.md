# 番茄短篇作品标签词表

生成短篇小说时，必须从下列词表中为作品选择标签，并写在正文最上方的「作品标签」区。只选择与本书核心卖点强相关的标签，不要堆满全表。

## 输出格式

```markdown
作品标签：
- 类型：〔1-3 个，见「类型」词表〕
- 角色：〔2-5 个，见「角色」词表；双男主或双女主作品必须在这里补充「双男主」或「双女主」〕
- 情节：〔2-5 个，见「情节」词表〕
- 情绪：〔1-3 个，见「情绪」词表〕
主角人设：〔人物 A 核心身份 / 处境〕 VS 〔人物 B 核心身份 / 性格 / 功能〕
```

「主角人设」必须使用 `A VS B` 的对照格式，例如：`不被爱的富家千金 VS 温柔暖心哥哥`。如果是无 CP 或群像，也要写成主角与主要对抗/陪伴力量的对照，例如：`破产法医 VS 连环旧案真凶`。
